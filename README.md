# Stack Exchange Activity Tracker

Automated tracking of Stack Exchange activity across all communities.

Data is fetched weekly from the [Stack Exchange API](https://api.stackexchange.com/docs) and organized by community under [`data/`](data/).

Each community directory contains:
- `README.md` — profile summary (reputation, badges)
- `answers.md` — all answers sorted by score
- `questions.md` — all questions sorted by score

Only communities with at least one question or answer are tracked.
